## Author

## Course
Web Design and Development

## Project Overview
**Zion’s Braid Bar — Crowned with Care** is a professional beauty and braiding website created as part of a Web Design course.  
The purpose of this project is to demonstrate foundational web development skills using HTML5 and CSS3, along with deploying a live website using GitHub Pages and testing locally with XAMPP.

The site highlights a clean, modern aesthetic suitable for a beauty brand and includes structured content, styling, and responsive layout principles.

## Technologies Used
- HTML5  
- CSS3  
- GitHub Pages (Live Hosting)  
- XAMPP (Local Hosting via Apache)  
- Visual Studio Code
